Save Changes to GitHub

Follow these steps to submit your changes as a Pull Request:

1
Copy your changes
Click the button to copy JSON with your changes to clipboard
2
Open GitHub editor
This will open the target file in GitHub's editor
3
Replace file contents
Select all text in the editor (Ctrl+A / ⌘+A) and paste your changes (Ctrl+V / ⌘+V)
4
Create Pull Request
Scroll down, select "Create a new branch", enter a description, and click "Propose changes"
